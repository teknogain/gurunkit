The `Card` component is a simple container with padding and border styling, used to group related content in a clean and structured layout.

## Base Component

This is the basic implementation of the card component:

```vue
<script setup>
defineProps({
  title: String,
  bordered: {
    type: Boolean,
    default: true
  },
  shadow: false,
  rounded: {
    type: Boolean,
    default: true
  }
})
</script>

<template>
  <div
    :class="[
      bordered ? 'border border-gray-300' : '',
      shadow ? 'shadow-md' : '',
      rounded ? 'rounded-md' : ''
    ]"
  >
    <div v-if="title" class="border-b border-gray-300 p-4 flex items-center justify-between">
      <h2 class="text-xl font-bold">{{ title }}</h2>

      <slot name="action" />
    </div>
    <div class="p-4">
      <slot />
    </div>
    <div v-if="$slots.footer" class="border-t border-gray-300 p-4">
      <slot name="footer" />
    </div>
  </div>
</template>
````

## Usage

You can use the `Card` component by importing and wrapping your content with it.

### Plain Card

```vue
<script setup>
import BaseCard from './components/base/base-card.vue'
</script>

<template>
  <base-card>
    Card Content
  </base-card>
</template>
```

This will render a box with rounded corners, a light border, and padding around the content inside.

### Header Title

```vue
<script setup>
import BaseCard from './components/base/base-card.vue'
</script>

<template>
  <base-card title="Card Title">
    Card Content
  </base-card>
</template>
```

This will render a card with a header section at the top, displaying the given title with a bottom border separating it from the main content.

### Header Title with Action

```vue
<script setup>
import BaseCard from './components/base/base-card.vue'
</script>

<template>
  <base-card title="User Info">
    <template #action>
      <button class="text-sm text-blue-600 hover:underline">Edit</button>
    </template>
    This card has a title and an action button in the header.
  </base-card>
</template>
```

This will render a card with a header that contains both a title on the left and a custom action button (like “Edit”) on the right, separated by space using a flexible layout.

### Footer

```vue
<script setup>
import BaseCard from './components/base/base-card.vue'
</script>

<template>
  <base-card>
    Card Content

    <template #footer>
      <div class="text-sm text-gray-500">
        Last updated 2 hours ago
      </div>
    </template>
  </base-card>
</template>
```

This will render a card with a footer section at the bottom, displaying additional content such as a timestamp or supplementary information, separated by a top border from the main content.

### Borderless Card

```vue
<script setup>
import BaseCard from './components/base/base-card.vue'
</script>

<template>
  <base-card :bordered="false">
    This card has no border.
  </base-card>
</template>
```

This will render a card without any border, creating a cleaner and more minimal look. It’s useful in layouts where the surrounding design already provides enough visual separation.

### Card with Shadow

```vue
<script setup>
import BaseCard from './components/base/base-card.vue'
</script>

<template>
  <base-card shadow>
    This card has a shadow effect.
  </base-card>
</template>
```

This will render a card with a subtle shadow effect (`shadow-md`), giving it visual depth and making it appear elevated above the background. It’s useful for highlighting important sections in the interface.

### Card without Rounded Corners

```vue
<script setup>
import BaseCard from './components/base/base-card.vue'
</script>

<template>
  <base-card :rounded="false">
    This card has no rounded corners.
  </base-card>
</template>
```

This will render a card with sharp corners (`rounded` disabled), giving it a more strict and rectangular appearance. It’s suitable for designs that require a more formal or technical look.

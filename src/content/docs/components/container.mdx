`Container` centers the content and adds horizontal padding based on screen size.

## Base Component

This is the basic implementation of the container component:

```vue
<template>
  <div class="container mx-auto px-4 sm:px-6 lg:px-8">
    <!-- Content -->
  </div>
</template>
```

## Usage

You can use the `Container` component by importing and wrapping your content with it.

```vue
<script setup>
import BaseContainer from './components/base/base-container.vue'
</script>

<template>
  <base-container>
    Container Content
  </base-container>
</template>
```
